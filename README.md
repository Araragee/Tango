# Tango - Vue 3 + TypeScript + Tailwind CSS

A premium, modern web application template scaffolded with Vue 3 (Composition API), TypeScript, and Tailwind CSS v4.

## Project Structure

```text
.
├── front-end/           # Frontend application
│   ├── src/             # Source code
│   ├── package.json     # Dependencies and scripts
│   ├── vite.config.ts   # Vite configuration
│   └── tsconfig.json    # TypeScript configuration
└── README.md
```

## Getting Started

Since this project was scaffolded in a restricted environment, you need to install the dependencies manually.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (v9 or higher)

### Installation

1. Navigate to the `front-end` directory:
   ```bash
   cd front-end
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Build for production:
```bash
npm run build
```

## Technologies Used

- **Vue 3**: Composition API with `<script setup>`.
- **TypeScript**: Static typing for better reliability.
- **Tailwind CSS v4**: The latest utility-first CSS framework with Vite integration.
- **Vite**: Ultra-fast build tool and dev server.
